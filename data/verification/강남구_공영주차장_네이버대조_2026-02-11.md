## 5. 데이터 품질 이슈

### 5-1. 이름 표기 문제
- 우리 데이터의 주차장 이름이 너무 축약되어 있음 (예: "수서역" → "수서역공영주차장")
- "공영주차장"이라는 접미사가 빠져 있어 일반인이 검색/인식하기 어려움
- 네이버 기준의 정식 명칭을 반영할 필요가 있음

### 5-2. 데이터 최신성 문제
- 일부 주차장의 `updatedAt`이 **2021-07-30** (약 4년 6개월 전 데이터)
  - 수서역, 학여울역, 일원역, 영동6교밑, 압구정고가1/2, 동호대교(남), 일원동, 일원터널
- 나머지 대부분은 **2024-07-05** 기준
- 2021년 이후 폐지되거나 변경된 주차장이 있을 가능성이 높음

### 5-3. 누락된 주차장 (네이버에 있으나 우리 데이터에 없음)
1. **역삼문화공원제1호공영주차장** (테헤란로7길 21) - 리뷰 722개로 인기 있는 주차장

- **구룡산제1호**와 **구룡산제2호**가 같은 주소 (양재대로 478), 같은 좌표 → 실제로 같은 장소에 2개가 있는지 확인 필요
- **일원동맛의거리**와 **대치유수지**가 같은 좌표 (37.504319, 127.0680509) → 좌표 오류 의심
- **신사역**과 **신구초교**가 같은 좌표 → 별도 주차장인지 확인 필요

- 대부분 `feeInfo`가 단순히 "유료"로만 되어 있어 사용자에게 유용한 정보 부족
- 네이버에서는 "5분당 320원" 같이 구체적 가격이 표시됨
- `basicTime`, `basicCharge` 등 숫자 필드는 있으나, `feeInfo` 텍스트가 불충분

## 6. 개선 권고사항

### 즉시 조치 필요
1. **누락 주차장 추가**: 네이버에서 확인된 6~7개 주차장을 우리 데이터에 추가
2. **좌표 오류 수정**: 동일 좌표를 공유하는 다른 주차장들의 좌표 검증
3. **2021년 데이터 갱신**: 4년 이상 된 데이터는 현재 운영 여부 확인 필요

### 중기 개선
4. **이름 정규화**: 네이버 기준 정식 명칭으로 통일 (예: "수서역" → "수서역공영주차장")
5. **요금 정보 상세화**: `feeInfo`에 "5분당 320원" 같은 구체적 정보 기입
6. **네이버 플레이스 ID 연동**: 이미 일부 데이터에 `naverPlaceId`가 있으나, 57개 전체에 확장 필요

### 장기 과제
7. **정기 검증 체계**: 월 1회 이상 네이버/카카오 대조 자동화
8. **다른 구 확장**: 강남구 외 나머지 24개 구도 동일 검증 필요

## 7. 검증 방법론 한계

- 네이버 지도 검색은 위치/관련성 기반 정렬이므로 모든 공영주차장이 결과에 나오지 않을 수 있음
- 노상주차장은 네이버 플레이스에 등록되지 않은 경우가 대부분이므로, 네이버 미노출이 곧 "존재하지 않음"을 의미하지는 않음
- 보다 정확한 검증을 위해서는 각 주차장을 개별 검색하거나, 강남구청/서울시설공단 공식 데이터와 대조하는 것이 권장됨
