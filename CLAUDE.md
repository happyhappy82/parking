# ParkingMap (parkingmap.kr)

## 서비스 목적

주차장별로 고유한 정보를 직접 조사하여 입력함으로써, 바이브코딩으로 카피 불가능한 우리만의 신뢰도 높은 사이트를 구축한다.

## 핵심 원칙

- 각 주차장에 대해 직접 검증한 고유 데이터를 축적한다
- 단순 공공데이터 복붙이 아닌, 차별화된 정보를 담는다
- 쉽게 복제할 수 없는 데이터 해자(moat)를 만든다

## 데이터 출처 정책

- **사용하는 데이터**: `data/공단주소/` — 각 구의 시설관리공단 홈페이지에서 직접 조사한 데이터
- **사용하지 않는 데이터**: `content/parking/` — 공공데이터포털(data.go.kr) 원본 데이터. 중복·오류가 많아 더 이상 사용하지 않음
- 데이터 출처는 오직 **각 자치구 시설관리공단** 기준이다
- 공공데이터포털 데이터는 보조 참고용으로도 쓰지 않는다
